# Thine Qualifier

A GTM tool built for [Thine](https://thine.com) — a personal intelligence app for the high-achieving elite.

## The Problem

Thine's growth lever isn't ads or content. It's getting the right 100 people using it, loving it, and talking about it publicly. But the current waitlist — comment on LinkedIn, drop your email — gives no signal on who's actually ready vs who's just curious.

## The Solution

A qualifier that thinks the way Thine thinks. 7 questions designed around how Thine's ideal user actually lives, works and makes decisions. Every submission scored, every person placed — Priority Access, On the Radar, Not Yet.

On the other side — an admin dashboard. Submissions grouped by date, ranked by score. Name. Email. Signal score. Who to reach first.

## Features

**User facing:**
- 7 question qualifier flow
- Smart scoring and signal-based tiering
- Duplicate submission prevention
- Name and email validation
- Result card with signal score out of 100

**Admin panel:**
- Password protected
- Submissions grouped by date, ranked by score within each date
- View answers toggle per submission
- Mark as Contacted — instant UI update, persists to Airtable
- Copy email button with tick feedback
- Delete submission
- Show/hide password + enter button
- Mobile friendly

## Tech Stack

- React + Vite
- Airtable — backend database
- Vercel — deployment
